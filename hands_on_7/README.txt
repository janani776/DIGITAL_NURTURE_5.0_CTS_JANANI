HANDS-ON 7 - ORM MIGRATIONS (ALEMBIC)

This folder demonstrates database migration concepts using SQLAlchemy ORM.

Contents:
- models.py includes updated schema with is_active column
- CourseSchedule table added to demonstrate schema evolution
- alembic configuration file included
- env.py links models to migration tool

Key Concepts:
- Database version control
- Schema evolution
- Migration tracking (upgrade/downgrade concept)

Note:
Actual execution of Alembic migrations is not performed.
Only structure and configuration are included for submission.
